# hrfax-frontend

hrfax-frontend is a cli tool for frontend develop

## Install

    $ [sudo]npm install -g hrfax-frontend-cli

## Usage 
    # 查看模板
    $ [sudo]hrfax list
    # 初始化项目
    $ [sudo]hrfax init

```
hrfax-frontend
```
